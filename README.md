Household Income Analysis using Python
Project Description

This project focuses on analyzing household income data using statistical techniques and data visualization in Python. The project includes descriptive statistics, measures of dispersion, probability distributions, skewness, kurtosis, and graphical analysis to understand income patterns and demographic relationships.

The dataset contains information related to:

Household income
Age
Education level
Family size
Urban/Rural areas

Python libraries such as NumPy, Pandas, Matplotlib, and Seaborn are used for statistical analysis and visualizations.

Objectives
Identify numerical and categorical data
Calculate central tendency measures
Analyze dispersion and variability
Study income distribution
Calculate skewness and kurtosis
Visualize data using statistical graphs
Compare household income across categories
